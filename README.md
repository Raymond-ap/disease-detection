# Disease Detection

This project aims to detect the presence of a particular disease in a patient using machine learning algorithms. The dataset used for training and testing the models is available in the 'data' directory.

## Dependencies

- Python 3.x
- scikit-learn
- NumPy
- Pandas
- Matplotlib
